# Careers Application

Built using: node.js@18, express, mongoDB, and mongoose.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn dev2

# watch mode
$ yarn watch

# production mode
$ yarn start
```
